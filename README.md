# nonebot_plugin_turbobot_completeform
#简介
Turbo Bot 是一个为 Turbo 用户提供指令服务的 Nonebot 插件。通过该插件，用户可以进行设置和管理用户功能票等操作。
#本插件是这个插件的扩展+重构 从Nonebot换为Nonebot2 添加了一些新的指令，并给所有指令都加入了详细说明
